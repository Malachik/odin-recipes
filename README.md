# Odin Recipes

This project is a recipe management application where users can create, view, and manage recipes. 

## Project Overview
The goal of this project is to practice and demonstrate skills in:

- HTML & CSS for building the user interface
- JavaScript for interactive functionality
- Git and GitHub for version control
- Problem-solving and project organization

## Learning Outcomes
By completing this project, I will have:

- Learned how to structure a web project from scratch
- Practiced front-end development skills
